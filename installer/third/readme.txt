To use Windows Inno setup installer to create an installer, this directory needs to contain:

From http://openssl.org
* openssl.exe
* libeay32.dll
* ssleay32.dll
* openssl.cnf

From http://nssm.cc
* nssm.exe

From  http://www.gzip.org/zlib/
* zlib.dll

From netcat ?
* nc.exe

And vcredist_x64-vc12.exe (for openssl) and vcredist_x64-vc12.exe (for uniproxy) matching the version of Visual Studio you are using.
