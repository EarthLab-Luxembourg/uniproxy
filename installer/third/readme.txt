To use Windows Inno setup installer to create an installer, this directory needs to contain:

From http://openssl.org
* openssl.exe
* libeay32.dll
* ssleay32.dll

From http://nssm.cc
* nssm.exe

From  http://www.gzip.org/zlib/
* zlib.dll

From netcat ?
* nc.exe

And vcredist_x86.exe matching the version of Visual Studio you are using.
